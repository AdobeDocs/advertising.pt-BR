# Exibir detalhes da simulação

*recurso do Beta*

Você pode exibir os detalhes da simulação na tela como um gráfico ou tabela.

1. No menu principal, clique em **[!UICONTROL Plan]>[!UICONTROL Simulations]**.

1. Clique no nome das simulações.

   A simulação é aberta na exibição de gráfico.

1. (Opcional) Na exibição de gráfico, siga um destes procedimentos:

   * Para alterar a métrica para a qual os dados estimados são mostrados para cada nível de meta de gasto (etapa), selecione uma nova métrica no menu **[!UICONTROL Y-Axis Metric]**.

   * (Somente simulações semanais) Insira um novo valor **[!UICONTROL Target Mid Point]** para ver os dados previstos usando esse valor.

   * (Somente simulações semanais) No menu **[!UICONTROL Constraints]**, escolha se os dados *[!UICONTROL With Constraints]*, *[!UICONTROL Without Constraints]* ou *[!UICONTROL With and Without]* devem ser exibidos.

1. (Opcional) Para abrir a exibição de tabela, clique na guia **[!UICONTROL Table View]**.

1. (Opcional) Para ver as configurações somente leitura do portfólio aplicável, clique em **[!UICONTROL Portfolio Settings]** no canto superior direito.
