# (Nova interface do usuário) Sobre a exibição [!UICONTROL Keywords]

*recurso do Beta*

A exibição [!UICONTROL Manage] > [!UICONTROL Keywords] inclui duas guias:

* A guia [!UICONTROL Keywords] lista todas as palavras-chave na exibição filtrada para a conta de anunciante selecionada.

* A guia [!UICONTROL Negatives] lista todas as palavras-chave negativas na exibição filtrada para a conta de anunciante selecionada.
