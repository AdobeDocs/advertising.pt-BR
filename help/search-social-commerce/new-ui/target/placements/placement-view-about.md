# (Nova interface do usuário) Sobre a exibição [!UICONTROL Placements]

*recurso do Beta*

A exibição [!UICONTROL Manage] > [!UICONTROL Placements] inclui duas guias:

* A guia [!UICONTROL Placements] lista todos os posicionamentos na exibição filtrada para a conta de anunciante selecionada.

* A guia [!UICONTROL Negatives] lista todos os posicionamentos negativos na exibição filtrada para a conta de anunciante selecionada.
