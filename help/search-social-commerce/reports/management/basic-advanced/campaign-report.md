# [!UICONTROL Campaign Report]

O [!UICONTROL Campaign Report] inclui dados de custo, clique e conversão (opcionalmente) de uma ou mais campanhas com palavras-chave. Por padrão, os dados incluem uma linha para cada campanha aplicável para cada unidade de tempo no intervalo de datas especificado, e as linhas estão em ordem crescente, primeiro por campanha e depois (quando aplicável) por portfólio.

Opcionalmente, você pode incluir classificações de rótulos específicas do anunciante.

Você pode exibir dados dos últimos 36 meses.
