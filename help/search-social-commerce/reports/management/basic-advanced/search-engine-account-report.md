# [!UICONTROL Search Engine Account Report]

O [!UICONTROL Search Engine Account Report] inclui dados de custo, clique e conversão (opcionalmente) de uma ou mais contas de rede de anúncios com campanhas. Por padrão, os dados incluem uma linha para cada conta aplicável para cada unidade de tempo no intervalo de datas especificado, e as linhas estão em ordem crescente, primeiro por rede de anúncios e, em seguida, por conta.

Você pode exibir dados dos últimos 36 meses.
