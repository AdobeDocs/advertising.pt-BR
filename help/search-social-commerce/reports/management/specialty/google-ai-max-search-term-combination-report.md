# [!UICONTROL Google AI Max Search Term Combination Report]

*Aplicável a contas [!DNL Google Ads] com campanhas habilitadas somente para AI max*

O [!UICONTROL Google AI Max Search Term Combination Report] mostra o desempenho de combinações de anúncios e páginas de aterrissagem específicas que o [!DNL Google Ads AI Max] usa com base em pesquisas na rede de pesquisa. O relatório inclui dados de impressão, cliques e custo para anúncios em [!DNL Google Ads] campanhas que usam [!DNL AI Max] em contas especificadas. Por padrão, os dados incluem uma linha para cada combinação de termo de pesquisa, título e página de aterrissagem que recebeu pelo menos uma impressão no intervalo de dados especificado. As linhas estão em ordem crescente por data e, em seguida, por campanha por padrão.

Use este relatório para ver como as consultas de pesquisa específicas são mapeadas para títulos gerados por IA e páginas de aterrissagem dinâmicas. Você pode usar os dados para analisar a intenção e o desempenho dos elementos de anúncio resultantes por consulta, de modo que possa criar listas de palavras-chave negativas robustas.
