# [!UICONTROL Google Performance Max Placement Report]

*Aplicável a contas [!DNL Google Ads] com campanhas de desempenho máximo somente*

O [!UICONTROL Google Performance Max Placement Report] inclui dados de impressão para posicionamentos em [!DNL Google Ads] campanhas de desempenho máximo em uma conta especificada. O relatório inclui as impressões para cada posicionamento incluído pelo URL de destino. Por padrão, os dados incluem uma linha para cada combinação de URL de posicionamento e direcionamento que recebeu pelo menos uma impressão no intervalo de dados especificado, mesmo que o posicionamento tenha sido excluído durante o período do relatório. As linhas estão em ordem crescente por campanha por padrão.

Use este relatório para:

* identifique quais sites, aplicativos e canais do [!DNL YouTube] estão veiculando seus anúncios.

* identifique e exclua posicionamentos irrelevantes ou com baixo desempenho para melhorar o ROI da campanha.

* certifique-se de que seus anúncios sejam exibidos apenas em ambientes seguros para a marca, criando listas de exclusão robustas.
